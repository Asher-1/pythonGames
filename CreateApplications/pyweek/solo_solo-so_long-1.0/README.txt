So long, a love story
=====================

Entry in PyWeek #7  <http://www.pyweek.org/4/>
Team: Solo solo
Members: riq

DEPENDENCIES:

You might need to install some of these before running the game:

  Python:     http://www.python.org/
  Pymunk:     http://code.google.com/p/pymunk/ (You might need chipmunk, but pymunk includes chipmunk)



RUNNING THE GAME:

On Windows or Mac OS X, locate the "run_game.pyw" file and double-click it.
Othewise open a terminal / console and "cd" to the game directory and run:

  python run_game.py



HOW TO PLAY THE GAME:

Move your "head" (blue ball) with the arrow keys.
And try to hit the "goal" ball (red ball) with your "tail" (red ball)
Your "head" shall not touch any "goal".
