===============
Heads and Tales
===============

An entry in PyWeek 14 by team Super Effective:
http://www.pyweek.org/e/nonstop_nidoran

This game requires AVBin (http://code.google.com/p/avbin/) to play music.
Without it the game will run but no music will play.

How to Play
===========

For source distributions run run_game.py. For binary distributions run the
application. Select 'New Game' at the main menu and enjoy!
