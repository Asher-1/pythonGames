Flight of the Pink Elephant
===========================

Entry in PyWeek #11  <http://www.pyweek.org/11/>
Code: Daniel Darabos ("cyhawk") <darabos.daniel@gmail.com>
Sounds and music: Beata Darabos ("bejus") <bejus79@gmail.com> & Daniel Darabos



DEPENDENCIES:

This game uses Python and Pyglet.

  Python:     http://www.python.org/
  Pyglet:     http://www.pyglet.org/



RUNNING THE GAME:

Open a terminal / console and "cd" to the game directory and run:

  python run_game.py



HOW TO PLAY THE GAME:

Use the arrow keys to move the elephant.
The goal is to save your wife from the cannibals.
Good luck!
