Lawn Fairy
===============

Entry in PyWeek #8  <http://www.pyweek.org/8/>
Team: Wood&Wolf
Members: Woodwolf (Programming) & Kokoro (Drawings, sound and music)


DEPENDENCIES:

You might need to install some of these before running the game:

  Python:     http://www.python.org/
    



RUNNING THE GAME:

On Windows or Mac OS X, locate the "run_game.pyw" file and double-click it.

Othewise open a terminal / console and "cd" to the game directory and run:

  python run_game.py



HOW TO PLAY THE GAME:

Move the fairy around the screen with cursor keys and blow with SPACE.

Blow bugs away before they eat all grass blades!
